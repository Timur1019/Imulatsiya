# Имулатсия — Киоск остановки

Симуляция двухэкранной системы:
- **Большой экран** — список приложений киоска
- **Экран 32"** — график автобусов + ротация рекламы
- **Админка** — настройка и публикация контента на экраны

## Запуск

### Backend (порт 8080)
```bash
cd backend
mvn spring-boot:run
```

### Frontend (порт 5173)
```bash
cd frontend
npm run dev
```

Откройте http://localhost:5173

## Экраны

| URL | Описание |
|-----|----------|
| `/` | Главная — навигация |
| `/admin` | Админ-панель |
| `/display/apps` | Большой экран — приложения |
| `/display/info` | 32" — автобусы + реклама |

## Как работает

1. В админке добавьте приложения и рекламные баннеры
2. Нажмите **«Опубликовать на экраны»** — данные сохраняются в БД
3. Экраны автоматически подхватывают изменения (polling каждые 5 сек)
4. Реклама крутится по кругу с заданной длительностью каждого баннера
5. Автобусы симулируются на бэкенде — движутся по маршрутам, ETA обновляется
